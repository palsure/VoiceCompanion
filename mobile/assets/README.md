# Assets Directory

Place your app assets here:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Quick Asset Generation

You can use online tools or create simple placeholders:

1. **Icon**: Create a 1024x1024px image with your app logo
2. **Splash**: Create a 1242x2436px image with your app branding
3. **Adaptive Icon**: Same as icon for Android
4. **Favicon**: 48x48px version of your icon

For now, you can use placeholder images or generate them using tools like:
- https://www.appicon.co/
- https://www.favicon-generator.org/
